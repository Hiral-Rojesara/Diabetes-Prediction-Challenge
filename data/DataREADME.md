# Dataset Information

This project uses publicly available diabetes datasets from Kaggle.

## Source Datasets

The following datasets were used during experimentation:

- diabetes_dataset.csv  
- diabetes_binary_5050split_health_indicators_BRFSS2015.csv  
- diabetes_binary_health_indicators_BRFSS2015.csv  
- diabetes_012_health_indicators_BRFSS2015.csv  
- train.csv (Playground Series S5E12)
- test.csv (Playground Series S5E12)
- sample_submission.csv (Playground Series S5E12)

## Kaggle Source

All datasets are sourced from Kaggle public datasets:
- Diabetes Health Indicators Dataset
- Playground Series S5E12

## Note

Due to size constraints, datasets are not uploaded in this repository.  
Users can directly download them from Kaggle and place them in this folder to reproduce results.
To reproduce results, download datasets from Kaggle and place them inside the `data/` folder.
