# Diabetes Prediction Challenge

This project focuses on predicting diabetes using machine learning with advanced feature engineering techniques. 
The aim is to transform raw medical and lifestyle data into meaningful health indicators that improve prediction accuracy.
Machine Learning project built for portfolio and real-world healthcare prediction use cases.

---

## 🚀 Project Overview

Diabetes is influenced by multiple factors such as blood pressure, cholesterol levels, BMI, age, and lifestyle habits.  
This project enhances raw data by creating engineered features that capture hidden health risks and relationships between variables.

---

## 🧠 Feature Engineering Pipeline

A custom feature engineering function is used to generate informative health indicators:

### 🔹 Blood Pressure & Heart Health
- Pulse Pressure
- Mean Arterial Pressure (MAP)
- Blood Pressure Ratio
- Hypertension Crisis Indicator
- Heart Stress Features

### 🔹 Lipid Profile
- LDL/HDL Ratio
- Triglycerides/HDL Ratio
- Non-HDL Cholesterol
- Total Lipid Index
- Cholesterol-Age Interaction

### 🔹 BMI & Physical Health
- BMI × Age Interaction
- Obesity Classification
- Waist-to-Hip Risk Features
- BMI Normalized by Age

### 🔹 Lifestyle Features
- Sleep vs Screen Time Balance
- Physical Activity Efficiency
- Stress vs Activity Ratio
- Sleep Efficiency Score

### 🔹 Statistical Features
- Age-based mean comparison features
- Deviation from age-group averages
